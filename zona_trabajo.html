Zona 3 — Área de trabajo (Dashboard + pantallas)

Esta es la zona donde realmente trabajas en NetSuite: puedes ver el Dashboard (Home), pero también formularios, búsquedas e informes. La pantalla cambia según lo que hagas.
← Volver

1) Cuando estás en Home: ves el Dashboard

Al dar clic en la casita (Home) vuelves a la página principal (inicio). Esa página principal es el Dashboard: un panel con información y accesos rápidos del rol.

¿Qué es un Portlet?

Un Portlet es cada bloque/tarjeta dentro del Dashboard. Cada portlet puede mostrar cosas distintas: recordatorios, KPIs, accesos directos, búsquedas rápidas, gráficos, etc.

Nota: para algunos roles el dashboard viene preconfigurado (no editable). Para otros roles, cada usuario puede personalizar qué portlets ve, en qué orden, y el diseño de columnas.

2) La Zona 3 cambia según lo que hagas

Esta misma zona se transforma cuando navegas: si usas un módulo del menú (Transacciones, Listas, Informes…), si buscas en el buscador o si abres un atajo, terminarás viendo formularios, listas/resultados o reportes.

Ejemplo A — Resultados (búsqueda / lista)
Pantalla típica de una búsqueda guardada o una lista: filtros arriba y tabla de resultados abajo. Aquí normalmente puedes ordenar, abrir registros y (según permisos) exportar.
Ejemplo B — Formulario (registro / transacción)
Cuando creas o editas algo, aparece un formulario: campos obligatorios, pestañas, sublistas y botones (Cliente, Oportunidad, Orden de venta, Factura, Caso, Tarea…).
Ejemplo C — Informe / Reporte
Cuando entras a Informes, verás reportes con tablas (y a veces gráficos), con filtros y opciones de exportación.
Idea clave: no importa si entras por el menú, por el buscador o por un acceso rápido: todo lo que haces se muestra aquí (Zona 3). Lo que cambia es la “vista” según el proceso.
