Navegación En netsuite

Al ingresar a NetSuite desde https://5055913.app.netsuite.com/ , la primera pantalla que verás es el Home, como se muestra a continuación. Esta vista se organiza en tres zonas principales: pasa el mouse para ver una explicación rápida y haz clic para ver el detalle. v3
Vista “macro”: 3 zonas generales. Al entrar al detalle, verás las sub‑partes (por ejemplo, el header se compone de 3 secciones).
